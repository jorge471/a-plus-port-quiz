# A+ Port Quiz Game

A simple Python quiz game that helps users memorize common networking ports for the CompTIA A+ certification exam.

## Ports Included

- HTTP (80)
- HTTPS (443)
- FTP (21)
- SSH (22)
- Telnet (23)
- DNS (53)
- SMTP (25)
- POP3 (110)
- IMAP (143)
- RDP (3389)

## Features

- Randomized questions
- Instant feedback
- Final score tracking

## How to Run

```bash
python port_quiz.py
```

## Skills Demonstrated

- Python fundamentals
- Lists and tuples
- Loops
- User input handling
- Basic networking knowledge
